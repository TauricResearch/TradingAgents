# Images Directory

This directory contains image files for the TradingAgents holdings system.

## Image Files

### portfolio-dashboard.png
Screenshot of the portfolio dashboard showing:
- Total portfolio value
- Gain/loss summary
- Holdings list with current prices
- Real-time updates

### holdings-chart.png
Visual representation of:
- Sector distribution (pie chart)
- Performance comparison (bar chart)
- Price trends (line chart)

### workflow-diagram.png
Illustration of the GitHub Actions workflow:
- Daily schedule triggers
- Data fetch process
- Update and commit steps
- Summary generation

## Usage in Documentation

Reference images in markdown files:

```markdown
![Portfolio Dashboard](../assets/images/portfolio-dashboard.png "Main Portfolio Dashboard")

![Holdings Distribution](../assets/images/holdings-chart.png "Sector Distribution Chart")

![Workflow Process](../assets/images/workflow-diagram.png "GitHub Actions Workflow")
```

## Adding New Images

1. Save image file to this directory
2. Use PNG, JPG, or SVG format
3. Optimize file size before committing
4. Update this README with description
5. Reference in project documentation

## Image Guidelines

- **Format**: PNG (recommended), JPG, or SVG
- **Size**: Keep under 500KB
- **Dimensions**: 1200x800px or similar
- **Quality**: High DPI for clarity
- **Naming**: Use lowercase with hyphens (e.g., portfolio-dashboard.png)
