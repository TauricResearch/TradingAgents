# Plans

Reusable planning prompts for architectural decisions, design debates, and trade-off analysis.
Use these before writing code — debate first, build second.

| Plan | Purpose |
|---|---|
| `arch-debate.md` | Debate two or more architectural approaches |
| `domain-design.md` | Design and validate a new DDD domain |
| `tech-choice.md` | Compare and decide between technologies or libraries |
| `api-design.md` | Debate and finalize an API contract |
| `refactor-strategy.md` | Plan a large refactor before touching code |
