# Skills Index

Each skill lives in its own directory. Invoke with `/skill-name [args]` in the chat.

| Skill | Invoke | Purpose |
|---|---|---|
| commit | `/commit` | Conventional commit from staged diff |
| review | `/review [file]` | Code review — Critical / Warning / Suggestion |
| test | `/test [file or function]` | Generate missing pytest tests |
| fix | `/fix [error or test name]` | Diagnose and minimally fix a bug |
| scaffold | `/scaffold <domain>` | Create full DDD domain structure |
| refactor | `/refactor [file]` | Improve structure without changing behavior |
| docs | `/docs [file]` | Generate Google-style docstrings |
| lint | `/lint [path]` | Run ruff + mypy and report issues |
| pr | `/pr [base-branch]` | Create a structured pull request |
| explain | `/explain [file or function]` | Plain-language explanation + DDD role |
